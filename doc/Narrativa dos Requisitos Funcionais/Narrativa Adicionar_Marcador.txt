Fluxo Principal

1.O Ator inicia o caso de uso
2.O sistema pede a localização do Marcador
3.O Ator fornece a localização do Marcador [fa01]
4.O sistema pede a foto para ser colocada no Marcador
5.O Ator fornece uma foto [fa02]
6.O sistema pergunta se o ator deseja adicionar um comentário
7.O Ator escreve, ou não.
8.O sistema adiciona o marcador no mapa de visualização
